Day 2: The PDA Loop
🎥 Video + 🖼️ Diagram ▼

Watch (30 min)

  • 🎥 ReAct Pattern Explained — Sam Witteveen
  • 🎥 DeepLearning.AI: Agent Loop — Lesson 2

Key Concepts

  • Perceive: Agent receives input (user query, tool results)
  • Decide: LLM reasons about what to do next
  • Act: Agent executes an action (call tool, respond)
  • The loop repeats until the task is complete

Activity (30 min)

  • Study the PDA Loop diagram (Diagrams tab)
  • Trace through the weather example step by step

  • RAG: Retrieval + generation, grounded answers
  • Agent: Tools + memory + planning + autonomy

🖼️ Diagrams

Visual guides for key concepts. All editable in Excalidraw.

1. The PDA Loop

How agents think: Perceive → Decide → Act, on repeat.

👁️ PERCEIVE 🧠 DECIDE ⚡ ACT Receive input → LLM reasons → Execute action User query Chain-of-thought Call tool, respond Tool results Plan next step Update memory 🔄 Loop until task complete Example: "What's the weather?" 👁️ User asks about weather 🧠 I need to call the weather API ⚡ Call weather_tool("Sydney") 👁️ Tool returns "25°C, sunny" 🧠 I have the answer ⚡ "It's 25°C and sunny in Sydney!"

2. Agent vs Chatbot vs RAG

Understanding the spectrum from simple chatbot to autonomous agent.

💬 CHATBOT 📚 RAG 🤖 AGENT ───────────────────────────────────────────────────────── User → LLM → Answer User → Search → LLM User → Think → Act → Loop ✅ Simple Q&A ✅ Knowledge retrieval ✅ Multiple tools ✅ Conversation ✅ Grounded answers ✅ Planning & reasoning ✅ Source citations ✅ Memory (short+long) ❌ No tools ✅ Autonomous actions ❌ No long-term memory ❌ No multi-step ✅ Multi-step tasks ❌ No planning ❌ Reactive only ❌ No autonomy Think: ChatGPT basic Think: Perplexity Think: Alex, AutoGPT Simple ←───────────────────────────────────────→ Powerful
✏️ Edit in Excalidraw

3. Tool Calling Architecture

How LLMs interact with external tools through structured JSON.

👤 User 🧠 LLM 📞 Tool Call "What's the weather?" → Reads tool schemas → { Decides: use search "tool": "weather", Generates JSON call "location": "Sydney" } ↓ 📋 Tool Schema 🔧 Tool Registry Available tools: ┌─────────┬──────────┐ - weather(location) │🔍Search │🧮Calculate│ - search(query) │💾Save │📧Email │ - calculate(expr) └─────────┴──────────┘ ↓ Results → back to LLM → Response to User Key: The LLM never runs tools directly — it outputs JSON that the system executes.
✏️ Edit in Excalidraw

4. Multi-Agent Collaboration

How specialized agents work together on complex tasks.

👑 COORDINATOR Routes tasks, checks quality ╱ │ ╲ ╱ │ ╲ 🔍 Researcher ✍️ Writer 📝 Editor Web Search Text Gen Grammar Check API Calls Formatting Fact Verify Data Analysis Templates Style Guide Pipeline Flow: 1️⃣ User Request → Coordinator receives topic 2️⃣ Coordinator → Researcher: "Find info about X" 3️⃣ Research findings → Writer: "Write about X" 4️⃣ Draft → Editor: "Polish this" 5️⃣ Final content → User Patterns: 🏢 Supervisor (above) 🤝 Peer (equals) 🔗 Pipeline (A→B→C) ⚖️ Debate (challenge)
