# Project 1: Agent Explorer 🔍

> Explore existing AI agents and understand how they work.

## Objective
Analyze 3 popular agent systems and compare their architectures.

## Steps (60 min)

### 1. AutoGPT (20 min)
- Visit [AutoGPT GitHub](https://github.com/Significant-Gravitas/AutoGPT)
- Read the architecture overview
- Watch a demo video

**Questions to answer:**
- What is AutoGPT's core loop?
- What tools does it have access to?
- How does it handle memory?
- What are its limitations?

### 2. BabyAGI (15 min)
- Visit [BabyAGI GitHub](https://github.com/yoheinakajima/babyagi)
- Read the README — it's surprisingly short!
- Understand the task creation → prioritization → execution loop

**Questions to answer:**
- How does BabyAGI break down tasks?
- How is it different from AutoGPT?
- What's the role of the "task creation" agent?

### 3. OpenClaw / Your Daily AI (15 min)
- Think about Alex (your AI assistant) — it IS an agent!
- What tools does Alex use? (calendar, email, search, code)
- How does Alex decide what to do?
- What makes it different from a chatbot?
